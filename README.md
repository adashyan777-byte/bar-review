# Бар — вправа на ревʼю коду

Тестове завдання для співбесіди.

## Структура

- `bar.ts` — система під тестом.
- `bar.test.ts` — юніт-тести (Vitest).
- `vitest.config.ts`, `tsconfig.json` — конфіги.

## Встановлення та запуск

```bash
npm install
npm test
npm run typecheck
```

Очікувана поточна поведінка: **4 тестів зелені** і **5 червоні**.
